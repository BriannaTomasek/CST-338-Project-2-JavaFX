# CST-338-Project-2
CST 338 Project 2, final project for the Return of the Java group with Ariya Brisco, Brianna Tomasek, Jessica Sandoval, and Vincent Marinello-Sweeney for Dr.C. 

Java Flash Trivia Application is an app to test your knowledge on Java by creating your own flash cards as a playable game. This game focuses on learning Java with customizable flash card questions to answer, with an accumulating score. After answering a question, the flash card will flip to the back, revealing the answer. If you get a question wrong your game has ended and you can choose to play again or go back to the main page. With multiple pages a user has access to many choices. On the main page you can log in with a user name/email and password, or sign up. The sign up page asks four standard questions to create a user. There is a page to search up users in case you’d like to delete and remove one. The start page gives you a choice to log out, play the game, or see results. On the welcome page you have choices to manage your questions or manage your users. The scores page shows the users “Today’s best”, “Overall Heighest”, and “Total plays”. Each answer you get correct leads you to the next flash card and question.  With multiple choice answers and a return to main menu button on each question, you can easily access where you want to go during the game. As you progress in Java, you can progress your questions too!
